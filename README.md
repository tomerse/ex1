#Recipe for strudel cake 
[Link for the cake's recipe][1]

##Ingredients:
1.  sugar
2.  oil
3.  3 eggs
4.  water
5.  280g flour
6.  5 plums

##How to make:
- blending
- add one egg
- add water, flour and plums
- blend again
- 45 minutes in oven, 170-180 degrees
